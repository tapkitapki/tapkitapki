Популярные места для рыбалки

Озеро Светлое

Популярное место для ловли щуки и окуня. Глубина до 8 метров, есть подводные ямы.

  • Виды рыб: щука, окунь, карась
  • Лучшее время: апрель-октябрь
  • Стоимость: бесплатно

Река Быстрая

Быстрая река с перекатами, идеальна для ловли голавля и жереха.

  • Виды рыб: голавль, жерех, язь
  • Лучшее время: май-сентябрь
  • Стоимость: бесплатно

Пруд Зеленый

Искусственный пруд с богатой ихтиофауной. Отлично подходит для семейной рыбалки.

  • Виды рыб: карп, карась, плотва
  • Лучшее время: май-сентябрь
  • Стоимость: 500 руб/день

Водохранилище Северное

Большое водохранилище с разнообразной рыбой. Есть оборудованные места для ловли.

  • Виды рыб: судак, лещ, сом
  • Лучшее время: июнь-сентябрь
